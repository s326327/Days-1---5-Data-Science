# 📊 Days 1–5 Data Science

A beginner-friendly Data Science learning repository that introduces core statistical concepts through hands-on Jupyter notebooks.

This repository is designed to build a strong foundation in data analysis by exploring how data is cleaned, summarized, interpreted, and analyzed using essential statistical techniques.

---

## 🚀 Learning Journey

### Day 1: Data is Messy
📄 `Lesson1_ Data is Messy.ipynb`

Learn about:
- Real-world data quality issues
- Missing values and inconsistencies
- Data cleaning fundamentals
- Preparing datasets for analysis

---

### Day 2: Three Ways to Describe Typical
📄 `Lesson2_ Three Ways to Describe Typical.ipynb`

Learn about:
- Mean
- Median
- Mode
- Measures of central tendency
- When to use each measure

---

### Day 3: How Spread Out is the Data?
📄 `Lesson3_ How Spread Out is the Data.ipynb`

Learn about:
- Range
- Variance
- Standard Deviation
- Data variability
- Interpreting spread in datasets

---

### Day 4: What Does the Shape of the Data Say?
📄 `Lesson4_ What Does the Shape of the Data Say_.ipynb`

Learn about:
- Data distributions
- Symmetric vs skewed data
- Histograms
- Understanding patterns in data

---

### Day 5: What Is Normal and Surprising?
📄 `Lesson5_ What Is Normal and Surprising_.ipynb`

Learn about:
- Normal distribution
- Probability concepts
- Outliers
- Detecting unusual observations
- Statistical reasoning

---

## ▶️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/s326327/Days-1---5-Data-Science.git
```

### Navigate to the Project

```bash
cd Days-1---5-Data-Science
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any notebook and run the cells to follow along with the lessons.
